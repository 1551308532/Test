# 实验盐值 OCR 分析应用

这是一个基于 Flask 的 Web 应用，用于对实验表格图片进行 OCR 识别，并自动完成以下任务：

- 提取表格中不同盐值下的各实验参数 Z 值；
- 计算每个盐值对应的 Z 值绝对值的总和、平均值、中位数与方差；
- 以实验组为横坐标，为最多五个参数绘制折线图，不同盐值使用不同颜色。 

## 使用方法

1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 启动应用：
   ```bash
   flask --app app run
   ```
3. 在浏览器中访问 `http://127.0.0.1:5000/`，上传包含盐值与 Z 值信息的表格图片，系统会返回统计结果与折线图可视化。

## 注意事项

- OCR 由 [EasyOCR](https://github.com/JaidedAI/EasyOCR) 提供，首次加载模型会花费一定时间；
- 请尽量提供清晰、光线均匀的表格图片，以便获得更准确的识别结果；
- 应用仅在识别到完整的“盐值”和“Z 值”模式时才会纳入统计。
